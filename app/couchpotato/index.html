COUCH POTATO

USER EXPERIENCE/INTERACTIVE/BRANDING

TOO BUSY TO KEEP UP WITH ALL YOUR FAVORITE SHOWS? DYING TO DISCUSS THE LATEST EPISODE OF SCANDAL?

For my Thinkful UX Design Immersive capstone project, I created a personalized television guide/viewing application that evolved from user research to sketching to prototyping. My goal was to create an easy-to-understand user interface flow that is also engaging and interactive. I aimed to make the colors of the interface bright and contrasting in order to add a youthfulness and freshness to the design as a whole. I used Marvel App to help create a realistic prototype of my application and it can be viewed here: http://marvl.in/ai8efe
